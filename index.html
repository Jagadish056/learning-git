Installation

Go to https://git-scm.com/download/win link for windows, download and install the git on your machine.

Git commands for beginners and their details

  1. git init

    The git init command creates a new Git repository. It can be used to convert an existing, unversioned project to a Git repository or initialize a new, empty repository.

  2. git config --global user.name 'FirstName LastName'
    git config --global user.email 'XXXX@XXXX.com'

    The first thing you should do when you install Git is to set your user name and email address.

  3. git status

    Displays the state of the working directory and the staging area. It lets you see which changes have been staged, which haven't, and which files aren't being tracked by Git. Status output does not show you any information regarding the committed project history..

  4. git add index.html
    git add folder
    git add .

    Displays the state of the working directory and the staging area. It lets you see which changes have been staged, which haven't, and which files aren't being tracked by Git. Status output does not show you any information regarding the committed project history..

  5. git rm --cached index.html
    git rm --cached folder

    Removes a file from the staging area. Opposite process of previous steps.

  6. git commit -m 'commit_messages'

    Used to move files from the staging area to a commit. This command is run after git add, which is used to add files to the staging area.

  7. git log

    shows a list of all the commits made to a repository. You can see the hash of each Git commit, the message associated with each commit, and more metadata.

  8. git checkout 145229ad305af5c8a44d48459b4cfbdd4e72d547(commit hash id)
    git checkout master
    git checkout other_branch

    Go to previous commit or master branch or any other branch.

  9. git branch new_branch(Only create new branch.)
    git checkout -b new_branch

    At first create the branch if not exits, then switch the branch

  10. git checkout master (Go to master branch)
    git merge new_branch

    Merge the Master branch to others branch

Connecting to your repository

  1. git remote
    git remote add origin https://github.com/username/repository_name.git

    Copy the repository url link from any github project

  2. git push -u origin master

    Connect your project to the github account. Sign in if required. Then, pushes to the repository. From the next time to push the project run commands git push

  3. git push

    Upload local repository content to a remote repository.

  4. git pull

    update the local version of a repository from a remote

Clone any github project

git clone https://github.com/username/repository_name.git
